# Proactive Resource Monitor

A lightweight, single-page dashboard for monitoring system resources and Clawdbot health.

## Features
- CPU & Memory usage visualization
- Disk space monitoring
- Clawdbot process status (via gateway API)
- Real-time updates

## Setup
1. Clone the repository.
2. Open `index.html` in a browser or serve via a lightweight web server.
3. Configure the gateway endpoint in `script.js` if necessary.
